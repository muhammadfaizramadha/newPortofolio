# Muhammad Faiz Ramadhan - Portfolio

Personal portfolio website showcasing my work as a web developer.

## Live Site

Visit: [itsfaiz.com](https://itsfaiz.com/)

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required

For development, use a local server:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000`

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Syne, Inter, JetBrains Mono (Google Fonts)
- **Form Service**: Formspree
- **Icons**: Simple Icons CDN

## Features

- Minimalist hero with scroll-driven animations
- Expandable experience timeline
- Filterable project portfolio
- Contact form with validation
- Responsive design (mobile-friendly)
- Scroll progress indicator
